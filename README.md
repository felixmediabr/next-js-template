# next-js-template
Next.js template for site creation

## Setup

### Dev
- TypeScript
- ESLint (airbnb)
- Prettier

## Instructions

### Development
- Run ```yarn```;
- Run ```yarn dev```;
- Browse to ```http://localhost:3000/```.

### Build
- Run ```yarn build```;
- Run ```yarn start```;
- Browse to ```http://localhost:3000/```.
